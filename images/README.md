# Product Images

This folder contains product images and the logo for the Fashion Store website.

## Logo

- **File**: `logo.jpg`
- **Usage**: Displayed in the header next to the "Fashion Store" text
- **Recommended size**: 200x200 pixels or similar square format
- **Format**: PNG with transparent background (preferred) or JPG

## Hero Background Image

- **File**: `hero-bg.jpg`
- **Usage**: Background image for the hero section at the top of the page
- **Recommended size**: 1920x1080 pixels (wide format)
- **Format**: JPG or PNG
- **Note**: The image will be displayed with a dark overlay to ensure text readability

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended size**: 800x800 pixels (square format works best)
- **File naming**: Use the naming convention `product1.jpg`, `product2.jpg`, etc.

## Placeholder Image

If an image is missing, the website will attempt to load `placeholder.jpg` as a fallback. You can add a generic placeholder image with this name for products that don't have images yet.

## Getting Images

You can:
- Use stock photos from sites like Unsplash, Pexels, or Pixabay
- Take your own product photos
- Use placeholder image services
- Create placeholder images using image editing software

## Tips

- Keep file sizes under 500KB for faster loading
- Use consistent lighting and background for a professional look
- Ensure products are centered and well-lit
- Consider using a white or neutral background
